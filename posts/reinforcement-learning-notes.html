Chapter 1: Introduction

Reinforcement learning is learning by interacting with our environment, like a child learns. There is an agent which interacts with environment which consists of states. Agent can take actions which can change environment state, perhaps in probabilistic fashion. Goal of the agent is to maximize rewards that it can get as it performs actions.

Reinforcement is third pillar of ML other than supervised learning, where correct actions for a state are provided and the model needs to learn them, and unsupervised learning which is about finding patterns in a data.

A policy (which is learnt during training) is mapping from state to action (can be probabilistic). Value of a state specifies the goodness of a certain state - which means how much (perhaps discounted) rewards can be collected from this state.

Historically, RL has had two threads. First thread originated in the psychology of animal learning. Second thread originated in optimal control and dynamic programming. Connection between optimal control and DP itself was slow to be realized.
